# Music Files

To enable background music on your PublicProfile page:

1. Add your music file to this directory as `Chill.mp3`
2. The music will auto-play at 40% volume when the page loads
3. Users can pause/play and adjust volume using the controls in the bottom-right corner

Supported formats: MP3, WAV, OGG
Recommended: MP3 files for better browser compatibility

Example: Place your music file here as `Chill.mp3`
